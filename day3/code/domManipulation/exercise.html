Experiment 1: Element Selecteren & Tekst Veranderen

Originele tekst - pas mij aan!
Stap 1: Selecteer het element
let element = document.getElementById('text-demo')
Stap 2: Bekijk het element
element

Je ziet het hele HTML element object in de console

Stap 3: Verander de tekst
element.textContent = 'Nieuwe tekst!'

De tekst verandert direct op de pagina!

Stap 4: Nog een keer
element.textContent = 'En nog een keer anders!'

Je kunt het oneindig vaak veranderen

Experiment 2: Kleuren & Styles Aanpassen

Stap 1: Selecteer de box
let box = document.getElementById('color-box')
Stap 2: Verander achtergrondkleur
box.style.backgroundColor = 'blue'

Box wordt blauw!

Stap 3: Meer kleuren proberen
box.style.backgroundColor = 'green'
box.style.backgroundColor = '#ff00ff'
box.style.backgroundColor = 'orange'
Stap 4: Andere styles aanpassen
box.style.width = '300px'
box.style.height = '300px'
box.style.borderRadius = '50%'

Nu is het een cirkel!

Stap 5: Transformaties
box.style.transform = 'rotate(45deg)'
box.style.transform = 'scale(1.5)'

Experiment 3: Elementen Verbergen & Tonen

Ik kan verdwijnen en terugkomen!
Stap 1: Selecteer element
let toggleEl = document.getElementById('toggle-demo')
Stap 2: Verberg element
toggleEl.style.display = 'none'

Element verdwijnt!

Stap 3: Toon element weer
toggleEl.style.display = 'block'

Element komt terug!

Stap 4: Probeer meerdere keren
toggleEl.style.display = 'none' // weg
toggleEl.style.display = 'block' // terug

Experiment 4: CSS Classes Toevoegen/Verwijderen

Element met classes
Stap 1: Selecteer element
let classEl = document.getElementById('class-demo')
Stap 2: Voeg class toe
classEl.classList.add('highlight')

Element krijgt nieuwe styling!

Stap 3: Nog een class
classEl.classList.add('shadow')

Element krijgt schaduw!

Stap 4: Verwijder class
classEl.classList.remove('highlight')
classEl.classList.remove('shadow')
Stap 5: Toggle class (aan/uit)
classEl.classList.toggle('highlight')

Run meerdere keren - het togglet!

Experiment 5: innerHTML vs textContent

Normale tekst
Verschil uitleggen:
  • textContent - Alleen tekst, geen HTML
  • innerHTML - HTML code die gerendered wordt
Stap 1: Selecteer element
let htmlEl = document.getElementById('html-demo')
Stap 2: textContent gebruiken
htmlEl.textContent = '<strong>Bold?</strong>'

Je ziet de HTML tags letterlijk!

Stap 3: innerHTML gebruiken
htmlEl.innerHTML = '<strong>Bold!</strong>'

Nu wordt het echt bold gerendered!

Stap 4: Meer HTML
htmlEl.innerHTML = '<em style="color: red">Rood en cursief!</em>'

Cheatsheet: DOM basics

Element selecteren:

document.getElementById('id') // Meest gebruikte document.querySelector('.class') // CSS selector document.querySelectorAll('.class') // Alle matches

Tekst aanpassen:

element.textContent = 'Nieuwe tekst' // Alleen tekst element.innerHTML = '<b>HTML</b>' // HTML wordt gerendered

Styles aanpassen:

element.style.backgroundColor = 'red' element.style.width = '200px' element.style.display = 'none' // Verbergen element.style.display = 'block' // Tonen

Classes:

element.classList.add('class-name') // Toevoegen element.classList.remove('class-name') // Verwijderen element.classList.toggle('class-name') // Toggle aan/uit element.classList.contains('class-name') // Check of class bestaat
